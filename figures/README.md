# Figures

Use vector PDF for plots and diagrams. A figure should make one comparison or
mechanism visible and its caption should state the takeaway. Keep labels at body
text size and use `reportblue` only for the focal result.

The report uses the annotated `*_triptych_extra_x` binding and hiding plots
from the ark-mt Criterion benchmark matrix. They show security thresholds and
the parameter points that require additional hash work.
